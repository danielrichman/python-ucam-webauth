python-raven - Ucam-webauth and Raven application agent in Python
=================================================================

Python dependencies
-------------------

* M2Crypto (`apt-get install python-m2crypto`; far easier than via pip)
* nose (for unit testing only)
* flask (for `simple_demo/`)
